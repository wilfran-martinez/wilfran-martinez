Programación Orientada a Objetos

Una clase, en el contexto de la programación orientada a objetos, es un concepto fundamental que se utiliza para definir un tipo de objeto con atributos y comportamientos específicos. Las clases son una forma de organizar y estructurar el código en programas informáticos, permitiendo la creación de objetos que comparten características y funciones comunes.

En términos más simples, puedes pensar en una clase como un plano o una plantilla para crear objetos. Una vez que has definido una clase, puedes crear múltiples instancias u objetos basados en esa clase. Estos objetos tendrán las mismas propiedades y métodos que se hayan definido en la clase.

Las clases constan de dos componentes principales:

  • Atributos o Propiedades: Son las características o datos asociados con la clase. Por ejemplo, si tienes una clase "Coche", sus atributos podrían ser "marca", "modelo" y "color".
  • Métodos: Son las funciones o acciones que pueden realizar los objetos creados a partir de la clase. Siguiendo el ejemplo del coche, los métodos podrían ser "arrancar", "detener" y "acelerar".

Ejemplo de Clase Coche en Java

A continuación se muestra un ejemplo de una clase en Java que representa un coche y sus métodos.

Código Java:




    public class Coche {
        // Atributos
        private String marca;
        private String modelo;
        private int añoFabricacion;
        private String color;
        private int velocidad;

        // Constructor
        public Coche(String marca, String modelo, int añoFabricacion, String color) {
            this.marca = marca;
            this.modelo = modelo;
            this.añoFabricacion = añoFabricacion;
            this.color = color;
            this.velocidad = 0; // El coche comienza sin moverse
        }

        // Métodos
        public void acelerar(int incrementoVelocidad) {
            velocidad += incrementoVelocidad;
            System.out.println("El coche ha acelerado a " + velocidad + " km/h.");
        }

        public void frenar(int decrementoVelocidad) {
            if (velocidad - decrementoVelocidad >= 0) {
                velocidad -= decrementoVelocidad;
                System.out.println("El coche ha frenado a " + velocidad + " km/h.");
            } else {
                System.out.println("El coche ya está detenido.");
            }
        }

        public void mostrarInformacion() {
            System.out.println("Marca: " + marca);
            System.out.println("Modelo: " + modelo);
            System.out.println("Año de Fabricación: " + añoFabricacion);
            System.out.println("Color: " + color);
            System.out.println("Velocidad: " + velocidad + " km/h");
        }

        // Método principal para probar la clase
        public static void main(String[] args) {
            Coche miCoche = new Coche("Toyota", "Corolla", 2022, "Rojo");

            miCoche.mostrarInformacion();
            miCoche.acelerar(50);
            miCoche.frenar(20);
            miCoche.mostrarInformacion();
        }
    }

Este es un ejemplo básico de una clase en Java que representa un coche y sus métodos. Puedes modificar y expandir este código según tus necesidades.
