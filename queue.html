Colas en Programación

En programación, una cola es una estructura de datos que sigue el principio FIFO (First In, First Out), lo que significa que el primer elemento que se inserta en la cola es el primero en ser eliminado.

En términos de implementación, una cola generalmente ofrece dos operaciones principales:

  • Enqueue (Encolar): Agrega un elemento al final de la cola.
  • Dequeue (Desencolar): Elimina y devuelve el elemento que está en el frente de la cola.

Además de estas operaciones básicas, algunas implementaciones de colas también pueden ofrecer operaciones adicionales, como Peek (para ver el elemento en el frente de la cola sin eliminarlo) o isEmpty (para verificar si la cola está vacía).

Ejemplo de Cola en Java

A continuación se muestra un ejemplo básico de cómo crear y manipular una cola en Java utilizando la clase Queue de la interfaz java.util.

Código Java:




  import java.util.LinkedList;
  import java.util.Queue;

    public class Main {
      public static void main(String[] args) {
        
        // Crear una cola
          Queue<String> cola = new LinkedList<>();

          // Añadir elementos a la cola
          cola.add("Elemento 1");
          cola.add("Elemento 2");
          cola.add("Elemento 3");

          // Mostrar la cola
          System.out.println("Cola: " + cola);

          // Eliminar y mostrar el primer elemento de la cola (FIFO)
          String elementoEliminado = cola.remove();
          System.out.println("Elemento eliminado: " + elementoEliminado);
          System.out.println("Cola después de eliminar: " + cola);

          // Obtener pero no eliminar el primer elemento de la cola 
          String primerElemento = cola.peek();
          System.out.println("Primer elemento de la cola: " + primerElemento);
          System.out.println("Cola después de peek: " + cola);
      }
  }

Este es solo un ejemplo básico para ilustrar cómo trabajar con colas en Java. Puedes modificar y expandir este código según tus necesidades.
