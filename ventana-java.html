Ventana basica en java

Este es solo un ejemplo básico para crear una ventana en Java. Puedes agregar más componentes a la ventana, como botones, etiquetas, campos de texto, etc., para crear interfaces de usuario más complejas.

Ventana:

Código Java:


    

    import javax.swing.*;
    import java.awt.*;

        public class Ventana extends JFrame {

            // Constructor de la clase Ventana
          public Ventana() {

            // Establece el título de la ventana
            setTitle("Sistema");

            // Establece la operación de cierre cuando se hace clic en el botón de cierre
            setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

            // Establece el tamaño de la ventana a 800x600 píxeles
            setSize(800, 600);

        // Centra la ventana en la pantalla
        setLocationRelativeTo(null);

        // Cargar la imagen del icono
        ImageIcon icono = new ImageIcon("img/icon-modified.png");
        Image imagen = icono.getImage();
        // Establecer la imagen como icono de la ventana
        setIconImage(imagen);
        }
        
        // Método main, punto de entrada del programa
        public static void main(String[] args) {

        // SwingUtilities.invokeLater se utiliza para asegurar que la creación de la interfaz de usuario se ejecute en el subproceso de despacho de eventos de Swing
        SwingUtilities.invokeLater(() -> {
       
        // Crea una instancia de la clase Ventana
        Ventana ventana = new Ventana();
        
        // Hace visible la ventana
        ventana.setVisible(true);
        });
        }
        }
